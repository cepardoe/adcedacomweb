PLATAFORMA DE CONTROL PARKING CAR “PARKTEC”

Es una herramienta para facilitar el servicio de parqueaderos: Controla entrada, recaudo y salida de vehículos. Puede implementarse a un lote de parqueo: privado, público, residencial, de un club social, un hotel, unidad empresarial, etc.

Caracteristicas funcionales

  1. Puede seleccionar la arquitectura que requiera: OnCloud, WEB, Cliente Servidor o Monousuario
  2. Permite controlar todos los tipos de vehículos que requiera: Camines, Tracto mulas, buses, vehículos normales, motos, o bicicletas
  3. Cierres de Caja por sitio, cajero y fechas
  4. Medidas de auditoria cuando se requiera: inventario de vehículos vs Facturación
  5. Inventario de vehículos y facturación
  6. Cierre de caja dependiendo del monto de dinero o al azar
  7. Manejo contable estricto
  8. Base de Datos robusta para archivos históricos
  9. Software desarrollado plataforma .NET
  10. Disminuye la perdida de dinero o de vehículos
  11. Diseño propio de tiquetes o facturas
  12. Permite integrarse a equipos como: barreras vehiculares, dispensadores, punto de pagos automáticos, validadores de salida, CTTV, Lectores Automáticos de Placa (LPR), Fidelización de Clientes

Beneficios

  1. Permite Cero perdidos de vehículos dependiendo el tipo de tecnología
  2. Controla 100% el recaudo del parqueadero
  3. Disminuye el fraude, el no cobro, cobros no establecidos, etc.
  4. Posee medidas de control no detectadas por el operario como: medidas persuasivas, medidas monitoreadas, medidas de investigación, etc.
  5. No requiere importar tecnología ni personal para mantenimiento, todos lo hacemos en Colombia y en su ciudad
  6. Sistemas desarrollados para mínimo 10 años de operación
  7. Disminución de gastos
  8. Aumento del Recaudo
  9. Seguridad al Conductor y vehículo

Licenciamiento

  1. Una Instancia de la Base de datos
  2. Software de Aplicación PARKTEC MULTIUSUARIO

Requerimientos de infraestructura

Servidor de base de datos
  1. Sistema operativo Microsoft Windows Server 2008 R2
  2. 100 GB disponibles en disco duro
  3. Memoria RAM 3 GB
  4. Procesador de 4 núcleos, 2,5 GHz por núcleo, TDP de 65 a 130W
  5. Microsoft SQL Server 2008 R2
  6. Desbloquear Firewall de Windows para SQL Browser, y puertos 1433 TCP (si es necesario).
Servidor de aplicaciones WEB
  1. Sistema operativo Microsoft Windows Server 2008 R2
  2. 100 GB disponibles en disco duro
  3. Memoria RAM 3 GB
  4. Procesador de 4 núcleos, 2,5 GHz por núcleo, TDP de 65 a 130W
  5. Internet Información Servicies 7.0
  6. Microsoft .NET Framework 4.0
  7. Habilitar el envío de correos electrónicos para este servidor
Requisitos para Cada Punto de Venta
  1. Computador con espacio disponible de 2 GB en Disco
  2. Instalación Módulo de Identificación Biométrica
  3. Una cámara USB de Mínimo 2 MP. En caso de no tenerla, está cotizada en el numeral de OFERTA ECONÓMICA
Requisitos Computador de Enrolamiento de Personas
  1. Computador con espacio disponible de 2 GB en Disco
  2. Instalación Módulo de Identificación Biométrica
  3. Puerto USB disponible para Lector de Huellas
Requisitos para configuración adicional del sistema
  1. Usuario y contraseña para acceder a los servidores, con privilegios de administrador
  2. Usuario y contraseña para SQL server, con privilegios de lectura, escritura, permitir copias de seguridad y conexiones remotas
  3. Correo electrónico dedicado para el envío de alertas vía e-mail, nombre, y contraseña que no caduque
  4. Nombre del servidor SMTP
  5. Número de puerto SMTP
  6. Requiere seguridad SSL (opcional)
