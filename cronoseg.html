PLATAFORMA DE CONTROL DE ACCESO, ASISTENCIA Y TIEMPOS PRODUCTIVOS

Es una solución para el control de acceso, asistencia y tiempos productivos de las empresas o entidades que contratan personal, la cual cuenta con lo último en avance tecnológico en: infraestructura, software, hardware, comunicaciones y servicios. El control de acceso se refiere a la verificación de las entradas, salidas, interrupciones de todas las personas que ingresan a las instalaciones de la organización. La asistencia es todo lo que tiene que ver con las personas ingresan a cumplir determinada labor o tiempo contratado.

Caracteristicas Principales

  1. Integra diferentes tecnologías de terminales, lectores o relojes Biométricos de marcas reconocidas en Colombia y de diferentes tecnologías biométricas como: Huella Dactilar, Mano, Venas, Iris, Rostro
  2. Agenda de Horarios Empleados automática sin tener que crear mallas de turnos En caso de no necesitar agenda de turnos CRONOSEG reconoce el turno por el perfil del empleado, fecha y marcaciones de entrada y salida
  3. Usuarios principales a la aplicación a través de cualquier explorador de internet: Gerentes, supervisores, auditores, encargados de nómina, directores de recursos humanos
  4. Disminución al 0% de ajustes manuales que puedan alterar registros
  5. 100% identificación de la persona que marca, 0% de perdida de datos, 100% de integridad de la información.
  6. Generación de alertas para: relojes desconectados, personal que llega tarde y personal ausente

Requerimientos de infraestructura

Servidor de base de datos
  1. Sistema operativo Microsoft Windows Server 2008 R2
  2. 100 GB disponibles en disco duro
  3. Memoria RAM 3 GB
  4. Procesador de 4 núcleos, 2,5 GHz por núcleo, TDP de 65 a 130W
  5. Microsoft SQL Server 2008 R2
  6. Desbloquear Firewall de Windows para SQL Browser, y puertos 1433 TCP (si es necesario).
Servidor de aplicaciones WEB
  1. Sistema operativo Microsoft Windows Server 2008 R2
  2. 100 GB disponibles en disco duro
  3. Memoria RAM 3 GB
  4. Procesador de 4 núcleos, 2,5 GHz por núcleo, TDP de 65 a 130W
  5. Internet Información Servicies 7.0
  6. Microsoft .NET Framework 4.0
  7. Habilitar el envío de correos electrónicos para este servidor
Requisitos para Cada Punto de Venta
  1. Computador con espacio disponible de 2 GB en Disco
  2. Instalación Módulo de Identificación Biométrica
  3. Una cámara USB de Mínimo 2 MP. En caso de no tenerla, está cotizada en el numeral de OFERTA ECONÓMICA
Requisitos Computador de Enrolamiento de Personas
  1. Computador con espacio disponible de 2 GB en Disco
  2. Instalación Módulo de Identificación Biométrica
  3. Puerto USB disponible para Lector de Huellas
Requisitos para configuración adicional del sistema
  1. Usuario y contraseña para acceder a los servidores, con privilegios de administrador
  2. Usuario y contraseña para SQL server, con privilegios de lectura, escritura, permitir copias de seguridad y conexiones remotas
  3. Correo electrónico dedicado para el envío de alertas vía e-mail, nombre, y contraseña que no caduque
  4. Nombre del servidor SMTP
  5. Número de puerto SMTP
